A) Overview - top

This is a jQuery plugin for dynamic cloud effect. It is simple and compatible with all major browsers and touches devices.

B) Included Files & Folders - top

The following files and folders are included in the package:

  • index.html - sample file. 
  • "\help" - this help document. 
  • "\js" Folder - contains the javascript files used. 
  • "\images" Folder - contains the image files used. 

C) Usage - top

Installing the plugin in your html page requires the following steps.

  •  Including the 2 required javascript files. 
  •  Specifying the html tags of the plugin. 
  •  Instantiating a javascript call with any optional parameters. 

Look at the the example in the index.html file.
Below describes the steps in more detail.

STEP 1. You will need to include the jquery's library file and the 3d shower's own 3dobjshow.js file. Both files are located inside the js directory: 

<script type="text/javascript" src="js/jquery.js"></script>
<script type="text/javascript" src="js/dynamiccloudeffect.js"></script>

You can also use the compressed version of the files:

<script type="text/javascript" src="js/jquery.min.js"></script>
<script type="text/javascript" src="js/dynamiccloudeffect.min.js"></script>

STEP 2. Set up your tags, give them an id or class: 

<div id="container"></div>

STEP 3, Call the plugin on $(document).ready():

<script type="text/javascript">
  $(document).ready
  (
    function() 
    {
      $("#container").dynamiccloudeffect 
      ({ 
          /* Parameters */
      }); 
    } 
  ); 
</script> 

               or call the plugin after the tag:

<div id="container"></div>
.....
<script type="text/javascript">
      $("#container").dynamiccloudeffect
      ({ 
          /* Parameters */
      }); 
</script> 

              


D) Parameters - top

You can change some common attributes of the effect through the parameters of the plugin. If you want to leave a parameter as is with its default value, you can omit that parameter in the javascript call.
Below is an example of the attributes being specified with corresponding values.

<script type="text/javascript">
  $(document).ready
  (
    function() 
    {
      $("#container1").dynamiccloudeffect
      ({ 
             full_screen:true,
             background_color:"#7090ef",
             background_image:"images/bg.jpg",
             cloud_number:26,
             speed:6,
             direction:"L-R"
      }); 
    } 
  ); 
</script> 

Another example:

<script type="text/javascript">
  $(document).ready
  (
    function() 
    {
      $("#container1").dynamiccloudeffect
       ({ 
                full_screen:false,
                screen_width:600,
                screen_height:500, 
                background_color:"#000000",
                cloud_number:26,
                cloud_size_zoom:1.5,
                cloud_images:"images/purple1.png;images/purple2.png;images/purple3.png",
                speed:3,
                initial_x:150,
                initial_y:-150,
                initial_disperse:3,
                direction:"T-RB"
    }); 
      $("#container2").dynamiccloudeffect
       ({ 
                full_screen:false,
                screen_width:"100%",              //pay attention to the quotation marks
                screen_height:350, 
                background_color:"#000000",
                cloud_number:20,
                cloud_size_zoom:2,
                speed:8,
                initial_disperse:1.5,
                direction:"L-R"
    }); 
    } 
  ); 
</script> 

parameters List Table:

Parameter Default Type Description
screen_width  600  Integer 
or String		 The width of the main container's window
or the width percentage of the width
screen_height  600  Integer 
or String		 The height of the main container's window
or the height percentage of the width
background_color  "#000000" Color  The color of the background
background_image  none String  The url of the of the background
full_screen false Boolean  Show the effect at full screen or not
cloud_number  26 Integer  The number of clouds in one plugin. The minimum is 8 and The maximum is 36
cloud_size_zoom  1.0  Real  The multiple of the zoom cloud size
speed  6  Real  The speed of the movement of clouds
direction "L-R"  String  The direction of the movement of clouds
"L-R": left to right
"R-L": right to left
"T-B": top to bottom
"B-T": bottom to top
"TL-BR": top left to bottom right
"BL-TR": bottom left to top right
"BR-TL": bottom right to top left
"TR-BL": top right to bottom left 
initial_x  -1  Integer  The x coordinates of the initial position of clouds
initial_y -1  Integer  The y coordinates of the initial position of clouds
initial_disperse  1 Real The disperse of the clouds at initial position
cloud_images "" String The url(src) of the cloud pictures. Each picture spare by ;
The string like:
"images/purple1.png;images/purple2.png;images/purple3.png"
Omitting the parameter is equal to setting the parameter to
"images/cloud1.png;images/cloud2.png;images/cloud3.png"
